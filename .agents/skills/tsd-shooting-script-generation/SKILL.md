# tsd-shooting-script-generation

**Status**: SCAFFOLD_ONLY

## Purpose

Repository skill scaffold for future shooting script generation.

## Inputs

- approved video strategy from concept strategy design

## Outputs

- core script
- shot-by-shot table
- English subtitles
- English voiceover notes
- Chinese review meaning
- shooting preparation list
- Hook variants

## Boundaries

- Does not create AI video assets.
- Does not invent unverified product effects.
- Does not replace human review.

## Failure Conditions

- shots cannot be executed
- visual proof cannot prove the claim
- subtitle depends on unverified claim
- information density exceeds target duration

## Evaluation Placeholder

Future eval cases should check factual accuracy, shootability, proof quality, and information density.
