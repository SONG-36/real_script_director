# repo-source-digest

**Status**: SCAFFOLD_ONLY

## Purpose

Repository maintenance skill scaffold for future Source Digest work.

## Inputs

- source material with metadata

## Outputs

- structured Source Digest

## Boundaries

- Does not scrape external materials automatically.
- Does not turn raw source text directly into knowledge.
- Does not bypass source metadata.

## Failure Conditions

- source is missing provenance
- source credibility or validity status is unclear
- source conflicts are not preserved

## Evaluation Placeholder

Future eval cases should check whether source conclusions remain traceable to source metadata.
